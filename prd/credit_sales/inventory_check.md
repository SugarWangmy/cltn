# 库存盘点模块需求文档

## 1. 产品概述

### 1.1 产品背景

库存盘点是美容门店产品管理的重要环节，用于核实实际库存与系统库存的一致性。本模块为移动端应用，支持员工通过扫码或手动输入方式进行库存盘点操作，确保库存数据的准确性和及时性。

### 1.2 产品目标

- 实现库存盘点流程的数字化管理，减少人工误差
- 支持多种盘点方式（扫码、手动输入），满足不同场景需求
- 实时记录盘点数据，便于后续分析和追溯
- 提高盘点效率，降低运营成本

### 1.3 适用范围

- 适用对象：美容门店员工、库存管理人员
- 使用设备：移动端设备（手机、平板）
- 使用场景：日常库存盘点、定期全面盘点、库存异常核查

---

## 2. 用户角色

| 角色 | 权限说明 |
|------|----------|
| 普通员工 | 执行盘点操作、扫描商品、确认盘点数据 |
| 店长 | 审核盘点结果、处理异常数据 |
| 管理员 | 创建盘点计划、查看所有盘点记录、系统配置 |

---

## 3. 功能需求

### 3.1 盘点流程管理

#### 3.1.1 创建盘点单

- 支持创建新的盘点单
- 可选择盘点类型：全面盘点、抽盘、随机盘点
- 可设置盘点范围：全部门店、指定门店、指定仓库
- 可设置盘点时间计划

#### 3.1.2 扫码入库

- 支持扫描商品条码快速识别商品
- 支持扫描箱码批量录入商品
- 扫描后自动显示商品信息（名称、规格、当前库存）
- 支持手动输入商品编码作为备选方式

#### 3.1.3 确认盘点

- 显示已扫描商品列表
- 支持修改盘点数量
- 支持标记差异商品
- 支持添加备注说明

### 3.2 商品管理

#### 3.2.1 商品识别

- 通过条码扫描识别商品
- 支持商品图片展示
- 显示商品基本信息（名称、规格、单位）
- 显示系统当前库存数量

#### 3.2.2 数量录入

- 支持手动输入盘点数量
- 支持使用加减按钮快速调整数量
- 自动计算盈亏数量和金额

### 3.3 数据展示

#### 3.3.1 盘点进度

- 显示已盘点商品数量
- 显示待盘点商品数量
- 显示盘点完成百分比

#### 3.3.2 差异显示

- 突出显示数量差异的商品
- 用颜色区分：盘盈（绿色）、盘亏（红色）、无差异（灰色）
- 显示差异数量和差异金额

### 3.4 盘点记录

#### 3.4.1 历史记录

- 展示历史盘点记录列表
- 显示盘点日期、盘点类型、盘点状态
- 支持查看盘点详情

#### 3.4.2 盘点报告

- 生成盘点汇总报告
- 显示盘盈盘亏汇总数据
- 支持导出盘点报告

## 4. 业务流程

### 4.1 标准盘点流程

```
开始 → 选择盘点类型 → 创建盘点单 → 扫码/输入商品 → 录入盘点数量 → 确认盘点 → 提交盘点数据 → 审核盘点结果 → 结束
```

### 4.2 流程节点说明

| 节点 | 操作说明 | 参与角色 |
|------|----------|----------|
| 创建盘点单 | 选择盘点类型和范围，创建盘点任务 | 管理员 |
| 扫码入库 | 扫描商品条码，识别商品 | 普通员工 |
| 录入数量 | 输入实际盘点数量 | 普通员工 |
| 确认盘点 | 核对盘点数据，确认提交 | 普通员工 |
| 审核结果 | 审核盘点数据，处理异常 | 店长/管理员 |

---

## 5. 界面原型

### 5.1 盘点首页

- 顶部：页面标题“库存盘点”，右侧快捷入口“输入条码”
- 步骤条：显示当前盘点进度（创建盘点单 → 扫码入库 → 确认盘点）
- 主体：显示待盘点任务列表或引导创建新盘点单

### 5.2 扫码页面

- 扫描区域：摄像头取景框
- 扫描结果：显示识别到的商品信息
- 操作按钮：“继续扫描”、“手动输入”

### 5.3 盘点确认页面

- 商品列表：显示已扫描商品
- 数量编辑：每行商品可编辑盘点数量
- 差异标注：自动计算并显示差异
- 底部操作：确认提交按钮

## 6. 非功能性需求

### 6.1 性能要求

- 页面加载时间不超过2秒
- 扫码响应时间不超过500毫秒
- 支持离线缓存，网络恢复后自动同步

### 6.2 兼容性要求

- 支持iOS 12.0及以上版本
- 支持Android 8.0及以上版本
- 适配主流屏幕尺寸

### 6.3 安全要求

- 用户登录验证
- 数据传输加密
- 操作日志记录
