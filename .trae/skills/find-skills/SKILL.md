# Find Skills

用于查找、管理和了解项目中的技能，帮助用户快速定位所需的技能资源，提高开发效率。

## 何时调用

- 需要查找特定功能的技能
- 了解现有技能的用途和使用方法
- 管理技能库，添加或更新技能
- 探索新的技能和工具

## 输出目标

- **技能列表**：提供符合条件的技能列表
- **技能详情**：展示技能的详细信息和使用方法
- **技能管理**：提供技能的添加、更新和删除功能
- **技能推荐**：根据用户需求推荐相关技能

## 技能分类

### 1) 开发技能

- 前端开发
- 后端开发
- 移动开发
-  DevOps
- 测试技能

### 2) 设计技能

- UI设计
- UX设计
- 视觉设计
- 交互设计

### 3) 业务技能

- 项目管理
- 产品管理
- 数据分析
- 市场分析

## 查找方法

1. **关键词搜索**：通过关键词查找相关技能
2. **分类浏览**：按技能分类浏览技能列表
3. **推荐系统**：根据用户历史使用记录推荐技能
4. **技能关联**：查看与当前技能相关的其他技能

## 技能管理

1. **添加技能**：创建新的技能并添加到技能库
2. **更新技能**：修改现有技能的信息和内容
3. **删除技能**：移除不需要的技能
4. **技能评估**：评估技能的有效性和使用频率

## 最佳实践

- 定期更新技能库，添加新的技能和工具
- 为技能提供清晰的描述和使用示例
- 建立技能分类体系，方便查找和管理
- 鼓励团队成员贡献和分享技能
- 定期评估技能的有效性和相关性
